Sobre o Projeto

Este projeto foi desenvolvido como parte da avaliação da unidade curricular de Tecnologias de Internet do 1º ano de Engenharia Informática.

O sistema de gestão de estacionamento inteligente utiliza tecnologias IoT para monitorizar em tempo real a ocupação de lugares, temperatura ambiente, luminosidade e outras variáveis relevantes para a operação eficiente de um parque de estacionamento.

A API RESTful desenvolvida segue os princípios SOLID, POO, DRY e implementa diversos padrões de design para garantir um código robusto, escalável e facilmente manutenível.

Principais Componentes:
  • Interface web responsiva
  • API para comunicação com dispositivos IoT
  • Sistema de autenticação seguro
  • Dashboard para monitorização em tempo real
  • Histórico de eventos e métricas
